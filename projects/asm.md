## Project Overview

**ASM® (Asset Management System)** is a comprehensive solution developed for the **ECT (Election Commission of Thailand) Central Office**. The system is designed to modernize and streamline the management of government assets, moving away from legacy paper-based or fragmented digital processes.

### Key Features

*   **Real-time Inventory Tracking**: Monitor asset locations, status, and assignment in real-time.
*   **Automated Depreciation**: Calculate asset value over time using standard government accounting rules.
*   **Audit Management**: Tools for periodic physical inventory checks and reconciliation.
*   **Reporting Dashboard**: Detailed insights into asset distribution, value, and lifecycle.

### Technical Implementation

The project was built using a robust enterprise stack:

*   **Backend**: ASP.NET Core with C# for high performance and type safety.
*   **Database**: Microsoft SQL Server for reliable data persistence and complex querying.
*   **Frontend**: Built with Angular to provide a structured, accessible interface for government officials.

### Impact

The implementation of ASM® has significantly reduced the time required for asset auditing and improved the accuracy of financial reporting for ECT.
