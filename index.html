VOTEMAI® – Decentralized Voting Application

A blockchain-based voting platform built using MERN Stack, Solidity, and IPFS. Ensures transparency and integrity through smart contracts on the MATIC (Polygon) network and secure wallet-based voting via Metamask. Developed and deployed smart contracts with Hardhat, integrated decentralized file storage using IPFS, and designed a responsive frontend using React.js and Bootstrap. Led the entire development process – coordinating frontend, backend, and blockchain integration for a seamless decentralized voting experience.

→

HOMEMA® – Technician Hiring Platform

Developed a web-based platform to connect users with qualified technicians. Key features include secure user authentication, full CRUD operations, and seamless RESTful API integration. Designed a responsive Bootstrap UI supporting mobile-first access, and deployed services using Render and Clever Cloud for scalability. Led a collaborative team as project leader, managing assignments, timelines, and development standards.
Tech stack: React.js, Node.js, MySQL, Bootstrap

ZSELL® – Point of Sale Web Application
